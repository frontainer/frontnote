Node.js StyleGuide Generator

FrontnoteはNode.js製のスタイルガイドジェネレータです。 

npm install frontnote --save-dev

Frontnoteについて

このスタイルガイドジェネレーターはKSSをインスパイアして開発されたものです。
KSSと同様にトップページにはMarkdownファイルを読み込み表示します。
KSSにない機能としてスタイル全体の概要、カラーパレット、スタイルに任意のラベル付けを有しています。

公式にGulp.js/Grunt.jsのプラグインもあり、簡単にスタイルガイド作成を自動化することができます。

テンプレートはカスタマイズ可能

テンプレートエンジンにEJSを採用しています。
既存のテンプレートを修正するだけでオリジナルのデザインでスタイルガイドを作成できます。
テンプレートのリポジトリを参照ください。

※ テンプレート作成ガイド鋭意制作中・・・

For Gulp.js

Gulp.js用のプラグインです。
gulp-frontnote

npm install gulp-frontnote --save-dev

※ gulp-frontnoteにはfrontnoteが含まれているので別途インストールの必要はありません

For Grunt.js

Grunt.js用のプラグイン　　 grunt-frontnote

※ grunt-frontnoteにはfrontnoteが含まれているので別途インストールの必要はありません
